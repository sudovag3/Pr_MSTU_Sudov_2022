§0. Корни нелинейных уравнений

 

Для численного решения задачи необходимо задать допустимую погрешность e.

Этапы нахождения корня:

1. Определяем диапазон, в котором находится единственный корень уравнения.

2. Находим корень уравнения с заданной допустимой погрешностью e.

Теоремы для общего понимания:

1. Теорема Больцано-Коши

    Если непрерывная функция, определенная на вещественном интервале, принимает два значения, то она принимает и любое значение между ними.

2. Следствие из теоремы Больцано-Коши

    Если функция непрерывна на некотором отрезке и на концах этого отрезка принимает значения противоположных знаков, то существует точка, в          которой значение функции равно нулю.


 

Рекомендуемые способы отделения корней

 

В вычислительной практике обычно используются следующие способы определения наличия корней, их количества и нахождения для каждого из корней достаточно малого отрезка [a, b], которому он принадлежит:

 

1) средствами машинной графики: функция f(x) представляется на дисплее и приближенно определяются отрезки, которым принадлежат нули функции;

2) средствами математического анализа с помощью исследования функций и построения графиков;

3) формированием простых функций  и  таких, что получается равносильное уравнение в виде , и дальнейшим построением графиков этих функций. Нулями функции в данном случае будут являться абсциссы точек пересечения функций f1(x) и f2(x). 
 

§1. МЕТОД БИССЕКЦИИ.

 

  1. Описание метода.

Дана функция f(x), являющаяся непрерывной на промежутке [a, b] и содержащая на нём одно нулевое значение. Дана допустимая погрешность e>0 с которой требуется вычислить приближенное значение x, чтобы выполнялось условие f(x)=0.

Находим число c - середину отрезка [a, b]. 

После нахождения c вычисляется значение f(c). Далее рассматриваем знаки значений f(a), f(b), f(c). Если f(a) и f(c) разных знаков, то за новое значение конца промежутка b принимаем значение c и повторяем действия описанные выше. Для ситуации, когда f(b) и f(c) разных знаков работаем аналогично.

       2. Критерий окончания.

Алгоритм повторяется до тех пор, пока не выполнится условие: разница между нынешним и предыдущим значениями c меньше заданной допустимой погрешности.

       3. Ограничения

Функция должна:

  • быть непрерывна на заданном промежутке [a, b]
  • содержать на промежутке [a, b] только один корень
  • удовлетворять условию f(a)*f(b)<0 (т.к. f(a) и f(b) разных знаков, и по первому ограничению функция непрерывна, то по следствию из теоремы Больцано-Коши на промежутке [a, b] существует точка, в которой значение функции равно нулю)

§3. МЕТОД ПРОСТЫХ ИТЕРАЦИЙ.

       1. Описание метода

Дана функция f(x), являющаяся непрерывной на промежутке [a, b] и содержащая на нём одно нулевое значение. Дана допустимая погрешность e>0 с которой требуется вычислить приближенное значение x, чтобы выполнялось условие f(x)=0.

Преобразуем функцию вида f(x)=0 к виду x=q(x) (т.е. выразим x). 

Выбираем начальное число c из промежутка [a, b] и подставляем в функцию x=q(x), получаем x=q(c).

Вычисляем значение x. Полученное значение можно обозначить как новое значение c, которые мы вновь подставляем в функцию.

       2. Критерий окончания

Производим выше описанные манипуляции до тех пор, пока разница между xn=q(xn-1) и xn+1=q(xn) не будет меньше допустимой погрешности.

Требования к данным

Входные данные:

Во всех методах: 1) левая часть уравнения от x - строка

                               2) точность вычисления корня Представляет собой число - десятичную дробь в виде: "0.0...01"

                               3) (Для методов Биссекции, Хорд, Касательных) границы отрезка функции - в интерфейсе обозначаются "a" и "b". Представляют                                           собой числа (как целые, так и вещественные). Могут быть представлены в виде дроби

                               4) (Для метода простой итерации) начальное приближение x0 - представляет собой число (как целое, так и вещественное). Может                                             быть представлено в виде дроби

 

Уравнение может содержать некоторые логарифмические функции, а именно:

синус -  при заполнении - "sin(x)"

косинус - при заполнении - "cos(x)"

тангенс - при заполнении - "tan(x)"

котангенс - при заполнении - "cot(x)"

Ур-е может содержать степенные функции

Знак степени в этом случае обазначается "**"

Ур-е может содержать дроби, знак деления в этом случае обознается - "/"

Выражения со скобками обособляются с обоих сторон символами "(" и ")" слева и справа соответственно

Знаки умножения, сложения, вычитания обозначаются символами "*", "+", "-" соответственно

Переменная "икс" обозначается символом "x"

Разделитель для дробных чисел = "."

 

ВЫХОДНЫЕ ДАННЫЕ

Общий вид всех выходных данных ПС при успешном вычислении корней: 

Название метода - строка (Например : “Метод биссекции”)

Введённое выражение - строка (Например : “ x**2 + sin(x) ”)

Численное значение корня - число (Например : 0.8687583054025084)

Точность - число (Например : 0.01)

Время на вычисление в секундах - число (Например : 0.01187570000001869)
